## Run Locally

Clone the project

```bash
  git clone https://github.com/SumitAthani/wedding
```

Go to the project directory

```bash
  cd invite
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Features


- Embedded google maps
